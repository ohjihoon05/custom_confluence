# PRD: 원익IPS Confluence 커스텀 매크로

## 배경 및 문제 정의

원익IPS SW팀은 장비 스펙, 공정 문서, 회의록 등 기술 문서를 관리할 내부 위키 플랫폼 도입을 검토 중이다. Confluence DC가 유력 후보이나, 기본 제공 매크로만으로는 문서 가독성과 구조화에 한계가 있다. Aura 같은 외부 앱은 라이선스 비용이 추가로 발생한다.

**핵심 문제**: 사내 맥락에 맞는 커스텀 매크로를 자체 제작할 수 있는가? 유지보수 가능한 수준인가?

## 목표

- Confluence DC 7.19.8 환경에서 Aura 수준의 커스텀 매크로 PoC 완성
- 자체 제작 시 외부 앱 라이선스 비용 절감 가능성 검증
- 사내 도입 결정 근거 마련

## 성공 기준

| 기준 | 측정 방법 |
|---|---|
| Page Title 매크로 정상 작동 | Confluence 페이지에서 매크로 삽입 → 렌더링 확인 |
| 편집 UI에서 파라미터 조작 가능 | Font Size, Color 변경 → 미리보기 반영 확인 |
| .jar 빌드 → 설치 → 동작 사이클 완성 | atlas-package → 앱 업로드 → 매크로 사용 |
| 비개발자도 매크로 사용 가능 | SW팀 비개발자 1인 테스트 통과 |

## 사용자

| 구분 | 대상 | 주요 행동 |
|---|---|---|
| 문서 작성자 | SW팀 엔지니어, 기획자 | 매크로 삽입, 파라미터 설정 |
| 관리자 | Confluence 관리자 | 매크로 설치, 업데이트 |
| 개발자 | 매크로 제작자 (현재 1인) | 빌드, 배포, 유지보수 |

## 핵심 기능

### F-01. Page Title 매크로 (PoC 1호)
- 현재 페이지 제목을 커스텀 스타일로 렌더링
- 설정 가능 파라미터:
  - `FontWeight`: Normal / Bold
  - `FontSize`: 12–96px (슬라이더)
  - `TextAlignment`: left / center / right
  - `Color`: HEX 색상값 또는 프리셋 스와치
  - `HtmlTag`: h1 ~ h6 (SEO 및 접근성 고려)
  - `LineHeight`: 수동 설정 토글

### F-02. 커스텀 편집 UI (P2 플러그인)
- iframe 기반 매크로 편집창 (Confluence 편집기 내 팝업)
- 좌측 파라미터 패널 + 우측 실시간 미리보기
- 구성 요소: 슬라이더, 정렬 버튼 3종, 컬러 스와치, 드롭다운

### F-03. 렌더링
- 페이지 뷰 모드에서 설정값 그대로 HTML 출력
- 페이지 로드 성능에 영향 최소화 (동기 렌더링)

## 비기능 요구사항

| 항목 | 요구사항 |
|---|---|
| 호환성 | Confluence DC 7.19.8 전용 |
| 편집 UI 로딩 | 2초 이내 |
| 렌더링 | 페이지 로드 추가 지연 100ms 이하 |
| 브라우저 | Chrome 최신, Edge 최신 |
| 접근성 | HTML Tag 선택으로 시맨틱 마크업 보장 |

## 에러 케이스 및 처리

| 상황 | 예상 동작 |
|---|---|
| FontSize에 문자 입력 | 기본값(48)으로 fallback |
| Color에 잘못된 HEX 입력 | 기본값(#000000)으로 fallback |
| 매크로가 페이지 외부에서 호출됨 | `$page`가 null → 빈 문자열 출력, 오류 미노출 |
| P2 플러그인 로드 실패 | Confluence 기본 에러 페이지 표시 (플러그인 외부 처리) |
| iframe 편집 UI 로드 실패 | 기본 파라미터 폼으로 폴백 (Confluence 기본 편집기) |
| Confluence 재시작 후 매크로 설정 유실 | DB에 저장된 파라미터 복원 (Confluence 기본 동작) |

## 엣지 케이스

| 케이스 | 처리 방법 |
|---|---|
| 페이지 제목이 빈 문자열 | `(제목 없음)` 텍스트 표시 |
| 페이지 제목에 HTML 특수문자 포함 (`<`, `>`, `&`) | Velocity `$esc.html()` 으로 이스케이프 |
| FontSize 극단값 (1px, 999px) | 렌더링은 허용, UI에서 12–96 범위로 슬라이더 제한 |
| 매우 긴 페이지 제목 (100자+) | CSS `overflow-wrap: break-word` 적용 |
| 다크 배경 페이지에서 기본 색상 | Color 파라미터로 사용자가 직접 조정 |
| 매크로를 템플릿에 포함 시 | 템플릿 기반 생성 페이지에서 정상 작동 (Velocity 변수 평가 시점 문제 없음) |

## MVP 제외 사항

- Confluence Marketplace 배포
- Confluence Cloud / 8.x 지원
- 다국어(i18n) 지원
- 매크로 2종 이상 (PoC는 Page Title 1종)
- 사용자 권한별 매크로 접근 제어
- 매크로 사용 통계 수집
